# Hand-Instrument-and-Physics-Simulations

## 1. Hand Instrument

Create amazing music by pinching your hands

Technologies used
* Unity 3d
* Simple Gesture Recognizer
* Oculus OVR Controller
* Oculus Hand trakcing
* Oculus Quest
